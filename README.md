# Personal Resource Hub (Interview Project)

A Vue 3 + Vite + TypeScript + Tailwind CSS base repo for recruiting take-home tasks.

## Getting Started

```bash
npm install
npm run dev
```

## Challenge Goals

1. Implement missing UI behavior (search, empty states).
2. Investigate a category filter bug (selected category does not update the list).
3. Improve LocalStorage persistence with safe serialization.
4. Complete TypeScript typing gaps and enforce type safety.
5. Keep layout responsive and accessible.

## Tech Stack

- Vue 3
- Vite
- TypeScript
- Tailwind CSS
- ESLint + Prettier
